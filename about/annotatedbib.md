# Annotated Bibliography

A citation that works as it should using this syntax: `` {cite}`Wandler.2017` `` produces {cite}`Wandler.2017`

And another citation that isn't working correcty using this syntax: `` {cite:p}`Wandler.2017` `` produces {cite:p}`Wandler.2017` but should show `[Wandler et al., 2011]`

And another citation that isn't working correcty using this syntax: `` {cite:t}`Wandler.2017` `` produces {cite:t}`Wandler.2017` but should show `Wandler et al., [2011]`
